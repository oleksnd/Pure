# PURE — Color Notes

**PURE** is a non-commercial hobby project dedicated to the study of color and its impact on human perception. It is not a technical library for developers, but rather a personal knowledge base and a collection of visual tools.

The project is hosted via GitHub Pages, providing a convenient way to access color data, analyze perception patterns, and explore color combinations.

### Project Overview
*   **Perception Study:** Analysis of how different colors influence emotions and psychological states.
*   **Research Collection:** A compilation of data and observations on color properties gathered from various sources.
*   **Visual Guides:** Composition rules, emotional palettes, and practical tips for color usage in design.
*   **Exploration:** An attempt to systematize information on design, physiology, and color psychology in one place.

### Usage
The project serves as a web-based reference tool for quick access to color characteristics and palette inspiration.
